11.1 选择符API

11.1.1 queryselector()

querySelector()方法接收一个css选择符，返回与该模式匹配的第一个元素。如果没有找到匹配的元素，返回null

11.1.2 queryselectorAll()方法

querySelectorAll()方法接收的参数与querySelector()方法一样，都是一个css选择符，返回的是一个NodeList的实例。调用querySelectorAll()方法的类型包括Document、DocumentFragment和Element。

11.1.3 matchesSelector()方法

Slector API Level 2规范为Element类型新增的一个方法，接受一个参数，即CSS选择符，如果调用元素与该选择符匹配，返回true，否则返回false
